# AI SEO Agent - Database Guide

## Overview

The AI SEO Agent stores competitor backlink data inside PostgreSQL.

The database is used to:

- Store competitors
- Store discovered backlink sources
- Store competitor backlink mappings
- Store backlink analysis results
- Store backlink opportunities
- Store email history
- Store workflow errors

The project uses PostgreSQL 17 running inside Docker.

# Restore Database

Copy backup

```bash
docker cp Database/seo_agent_db.backup seo-postgres:/tmp/seo_agent_db.backup
```

Restore

```bash
docker exec -it seo-postgres pg_restore -U postgres -d seo_agent_db --clean --if-exists -v /tmp/seo_agent_db.backup
```

# Database Tables

## competitors

Purpose

Stores all monitored competitors.

Columns

| Column | Description |
|---------|-------------|
| id | UUID Primary Key |
| company_name | Company Name |
| domain | Website Domain |
| industry | Industry |
| status | ACTIVE / INACTIVE |
| created_at | Created Time |
| updated_at | Updated Time |

Example

```
InfiniApps

Microsoft

Zoho
```

---

## backlink_sources

Purpose

Stores unique backlink source websites.

Each domain exists only once.

Example

```
github.com

medium.com

linkedin.com

dev.to
```

---

## competitor_backlinks

Purpose

Maps competitors to backlink sources.

Relationship

```
One Competitor

↓

Many Backlinks
```

Contains

- Source URL
- Anchor Text
- First Seen
- Last Seen
- Analysis Status

---

## backlink_analysis

Purpose

Stores AI analysis generated by Agent 2.

Contains

- Domain Rating
- Relevance Score
- Authority Score
- Spam Score
- Opportunity Score
- Recommendation

---

## backlink_opportunities

Purpose

Stores high-quality backlink opportunities generated by Agent 3.

Marketing team uses this table.

---

## email_logs

Purpose

Stores all sent reports.

Useful for auditing.

---

## error_logs

Purpose

Stores workflow failures.

Useful for debugging.

# Useful Queries

List competitors

```sql
SELECT *
FROM competitors;
```

Only active competitors

```sql
SELECT *
FROM competitors
WHERE status='ACTIVE';
```

Latest backlinks

```sql
SELECT *
FROM competitor_backlinks
ORDER BY created_at DESC;
```

Latest opportunities

```sql
SELECT *
FROM backlink_opportunities
ORDER BY created_at DESC;
```

Latest errors

```sql
SELECT *
FROM error_logs
ORDER BY created_at DESC;
```
